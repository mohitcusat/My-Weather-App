# 🌦️ Weather App

A modern and responsive Weather Application built using React, Vite, and Tailwind CSS. The application allows users to search for any city and view real-time weather information with a clean and user-friendly interface.

## 🚀 Features

- Search weather by city name
- Real-time weather updates
- Temperature, humidity, and wind speed information
- Responsive design for desktop and mobile devices
- Fast and optimized performance with Vite
- Clean and attractive user interface

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Weather API

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/harsh123sinha/my-weather-app.git
```

### Navigate to Project Directory

```bash
cd my-weather-app
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory and add your API key:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

### Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:5173
```

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
